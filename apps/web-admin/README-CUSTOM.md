# Pharmacy Admin Portal

Admin portal for the Multi-Vendor Pharmacy Platform. Built with Next.js 14, TypeScript, and Tailwind CSS.

## Quick Start

```bash
# Development
npm run dev --workspace=web-admin

# Build
npm run build --workspace=web-admin

# Production
npm run start --workspace=web-admin
```

## Features

- ✅ Responsive dashboard with real-time statistics
- ✅ Revenue chart visualization
- ✅ Recent orders tracking
- ✅ User-friendly navigation

## Access

- Development: http://localhost:3000
- Dashboard: http://localhost:3000/dashboard
