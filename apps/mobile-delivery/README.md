# Delivery Personnel Mobile App

Delivery personnel mobile application for the Multi-Vendor Pharmacy Platform built with React Native and Expo.

## Features

- **Delivery Management**: View and manage assigned deliveries
- **Real-time Tracking**: GPS navigation to customer locations
- **Status Updates**: Update delivery status in real-time
- **Order Details**: View order information and customer details
- **Performance Metrics**: Track deliveries and earnings
- **Notifications**: Receive delivery assignments and updates

## Tech Stack

- React Native 0.73
- Expo SDK 50
- TypeScript
- React Navigation v6
- Expo Location for GPS tracking
- Async Storage

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Run on Platform

```bash
# iOS
npm run ios

# Android
npm run android
```
