# Vendor Mobile App

Vendor management mobile application for the Multi-Vendor Pharmacy Platform built with React Native and Expo.

## Features

- **Dashboard**: Overview of sales, orders, and inventory
- **Inventory Management**: Track stock levels and manage products
- **Order Fulfillment**: Process and manage customer orders
- **Sales Analytics**: View sales reports and performance metrics
- **Expense Tracking**: Track business expenses
- **Notifications**: Receive order alerts and updates

## Tech Stack

- React Native 0.73
- Expo SDK 50
- TypeScript
- React Navigation v6
- Async Storage

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Run on Platform

```bash
# iOS
npm run ios

# Android
npm run android
```
